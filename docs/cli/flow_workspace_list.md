## flow workspace list

List all registered workspaces.

```
flow workspace list [flags]
```

### Options

```
  -h, --help              help for list
  -o, --output string     Output format. One of: yaml, json, or tui. (default "tui")
  -t, --tag stringArray   Filter by tags.
```
