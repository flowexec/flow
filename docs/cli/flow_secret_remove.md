## flow secret remove

Remove a secret from the vault.

```
flow secret remove NAME [flags]
```

### Options

```
  -h, --help            help for remove
  -o, --output string   Output format. One of: yaml, json, or tui.
  -V, --vault string    Vault name to use instead of the current vault.
```
